# robot_chien_-quipe_41

Code for controlling a robot dog using Arduino.

## Purpose
This project is intended as a starting point for users who want to experiment and modify the robot’s behavior.

## Modifying Behavior
Users can change the code to:
- Adjust movement behavior
- Modify timing or motion sequences
- Add new functions
- Test different control ideas

Review the main code and comments before making changes.

## Uploading Changes
1. Open the code in Arduino IDE  
2. Modify the behavior you want to test  
3. Upload the updated code to the robot  
4. Test changes carefully
